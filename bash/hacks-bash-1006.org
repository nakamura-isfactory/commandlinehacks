$ vim /etc/fstab  <Ctrl+U>
#+END_EXAMPLE

$

$ sudo <Ctrl+Y>

#+BEGIN_EXAMPLE
$ sudo vim /etc/fstab
#+END_EXAMPLE
* 用法
Ctrl+U → コマンドラインにある文字列を全てキルする。
Ctrl+Y → キルした文字列をヤンクする。
